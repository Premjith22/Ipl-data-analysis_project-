# 🏏 IPL Data Analysis Project

## 📌 Project Overview
This project analyzes IPL (Indian Premier League) data from 2008–2024 using Python libraries like Pandas, NumPy, Matplotlib, and Seaborn.

The project focuses on:
- Team performance analysis
- Batsman analysis
- Bowling analysis
- Toss impact analysis
- Venue analysis
- Match trends
- Data visualization

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## 📊 Key Insights
- Mumbai Indians and Chennai Super Kings are among the most successful teams.
- Virat Kohli is among the leading run scorers.
- Toss decisions impact match outcomes.
- IPL matches increased significantly over the years.

---

## 📂 Dataset
Dataset used:
- matches.csv
- deliveries.csv
